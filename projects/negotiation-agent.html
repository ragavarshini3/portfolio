AI Negotiation Agent

A multi-stage dialog system using natural language processing pipelines and intent classifications to simulate dynamic negotiations.

The Challenge

Commercial and legal negotiations are inherently complex, requiring players to understand sentiment, context, and concession thresholds. Standard chatbots built on deterministic state-trees cannot handle the flexibility needed for active bargaining or pricing concessions.

The objective was to create an intelligent conversational agent that could negotiate items dynamically, interpret user intents (e.g. counter-offer, compromise, rejection), map conversations across multi-stage negotiation cycles, and finalize optimal agreements.

The Solution

I designed an AI Negotiation Agent written in Python. The system processes user messages using Hugging Face Transformer models to extract text embeddings, performing intent analysis and sentiment scoring. A secondary rule-based strategy layer maintains session state (Initial Offer, Bargaining, Concession, Agreement/Deadlock) and evaluates pricing math, ensuring that concessions do not breach a pre-configured reservation price while adjusting counter-offers dynamically to mimic human bargaining styles.

Key Challenges & Resolutions

  • Context Drift in Long Chats: Conversation history can quickly pollute model contexts, leading to agent loop repeats. I resolved this by utilizing a sliding-window memory buffer that summarized preceding negotiation rounds and logged numerical values separately in a structured database schema.
  • Validating Complex Offers: Free-text pricing offers are difficult to parse. I built regex validation layers coupled with tokenizers to extract numeric patterns (e.g. "let's do $500 instead") and map them to structural variables.
  • Balancing Math and Naturalness: Purely mathematical models feel rigid. I combined structured strategic limits (minimum threshold logic) with conditional response templates, enabling the bot to assert, compromise, or push back in a natural tone.

Project Outcomes & Results

  • Successfully developed a functional, multi-state negotiation bot simulation.
  • Created structured tokenizers parsing numeric offers and conversational intents.
  • Established clear boundaries to prevent agents from accepting unprofitable terms.
  • Deepened understanding of NLP workflows, context window management, and conversational systems.
